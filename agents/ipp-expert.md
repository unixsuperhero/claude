- **No `import React from 'react'`** â€” project uses automatic JSX runtime (`react-jsx`); only import named exports like `{ useState }`

- **Router files are critical-path bundles** â€” use `React.lazy()` for component references, never static imports of UI components
- **Permissions**: Use `DomainAccessGated` for access control

**Reading data:**
